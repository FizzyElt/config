# Config

This is a configuration repository for my dotfiles. It contains various configuration files for different applications and tools that I use on a daily basis. The purpose of this repository is to keep all my configurations organized and easily accessible.

```bash
npx @fizzyelt/config ocaml
npx @fizzyelt/config oxc
npx @fizzyelt/config oxc -t
```

- OCaml
  - `ocamlformat`
- TypeScript
  - `Oxlint`
  - `Oxfmt`
